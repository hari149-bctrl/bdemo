Privacy Policy

Effective Date: 13/05/2025

This Privacy Policy explains how Brainy Voyage Instabot ("we", "our", or "us") collects, uses, and protects information obtained through our platform, including data accessed via Meta (Facebook/Instagram) APIs.

1. Information We Collect

With user consent, we collect through Meta's APIs:

  • Facebook Page information (ID, name, insights)
  • Instagram Business Account data (username, follower counts, profile picture)
  • Post metadata and comments from Instagram
  • Message content (inbound/outbound) via Instagram Direct Messaging

2. How We Use Your Data

All data is used solely to:

  • Display and manage content within our dashboard
  • Enable comment moderation (reply/hide/delete)
  • Send automated Direct Message responses
  • Provide engagement analytics
  • Improve app functionality

We never use data for: Advertising, cross-site tracking, or building user profiles for unrelated purposes.

3. Data Sharing & Storage

We do not sell or rent Platform Data. Limited sharing occurs with:

  • MongoDB Atlas - Secure database storage (under Data Processing Agreement)
  • GitHub - Code version control (no Platform Data stored)

All providers are contractually prohibited from independent data use and maintain equivalent security standards.

4. Data Retention

Platform Data is:

  • Cached for up to 7 days for real-time functionality
  • Automatically purged thereafter
  • Retained longer only when legally required

5. Security Measures

We implement:

  • HTTPS encryption for all data transfers
  • AES-256 encryption for data at rest
  • Strict access controls with role-based permissions
  • Regular security audits

6. Your Rights

You may:

  • Access/Export: Request your data in machine-readable format
  • Delete: Remove all stored data by contacting us
  • Revoke Access: Disconnect via Facebook/Instagram settings

All requests are processed within 7 days.

7. Meta Compliance

We strictly adhere to:

  • Meta's Platform Terms and Developer Policies
  • Data Minimization Principles (only collecting what's necessary)
  • Annual security reviews of our Meta integration

8. Government Requests

If compelled to disclose data, we:

  1. Verify the request's legality
  2. Disclose only the minimum required information
  3. Notify affected users unless legally prohibited

We have never received such requests to date.
